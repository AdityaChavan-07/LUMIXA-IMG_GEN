Lumixa is a self-hosted AI image generation studio — describe what you want, and watch it render. Built as a lightweight Flask backend paired with a dependency-free vanilla JavaScript frontend, it's designed to feel like a polished commercial product without any of the usual overhead: no React, no build step, no bundler, and no GPU required on your end.
Under the hood, Lumixa talks to Pollinations.ai, a free and open image generation API, so you can go from git clone to your first generated image in minutes — no signup, no API key, no billing setup. An optional API key is supported if you outgrow the free tier's rate limits.
The interface takes cues from modern AI product design: a dark, glassmorphic aesthetic with violet/cyan neon accents, a signature animated gradient border on the prompt console, and buttery CSS-only micro-interactions throughout. Beyond the core "type a prompt, get an image" loop, Lumixa includes the features you'd expect from a real product — a searchable, date-grouped history; a favorites collection; a Pinterest-style masonry gallery; a fullscreen viewer with zoom, pan, and keyboard shortcuts; per-image actions like download, share, copy-prompt, and regenerate; and a full settings panel for theming and generation defaults.
Everything beyond the actual image request — history, favorites, gallery, and preferences — lives entirely in the browser via localStorage, keeping the backend intentionally thin: two routes (/generate and /health) and nothing else to deploy, configure, or maintain.
Whether you're prototyping a generative-AI product, learning how to wire a frontend to an image API, or just want a nice-looking personal image generator you can run locally, Lumixa is meant to be a clean, hackable starting point.
